linuxpedia_diagrams
===================

This repository contains the orginal project files (vsd,ppt,odf etc) of the 
diagrams (in format png, gif, jpg, jpeg) used at linuxpedia.org. This enables 
users to easily update the diagrams by editing the orginal project file and 
converitng it to png, gif, jpg or jpeg format.

For any help or queries please contact admin@linuxpedia.org.
